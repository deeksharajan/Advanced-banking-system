#  Advanced Banking System (C++)

A simple console-based **Banking System** built using C++ and Object-Oriented Programming (OOP).  
This project simulates real-world banking operations such as login, balance enquiry, deposit, withdrawal, and transaction history.

---

##  Features

-  Secure PIN-based authentication  
-  Check account balance  
-  Deposit money  
-  Withdraw money  
-  View transaction history  
-  Menu-driven interface  
-  Beginner-friendly OOP implementation

# How It Works

* User starts the program
* System asks for PIN authentication
* After successful login, menu is displayed
* User can perform banking operations:
* Check Balance
* Deposit Money
* Withdraw Money
* View Transaction History
* Program continues until user selects Exit
